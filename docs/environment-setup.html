Liferay IDE

Download Liferay IDE at : http://www.liferay.com/downloads/liferay-projects/liferay-ide. This Eclipse-based IDE will help you to deploy your portlet applications in a pre-configured application server. Debugging will be also easier this way.

Installing a few plugins before starting is recommended :

  • m2e (Maven plugin) and EGit (Git plugin), availables from Juno update site (http://download.eclipse.org/releases/juno)
  • Web Tools for Tapestry, available at https://github.com/got5/webtools-tapestry

    • In this tutorial, we will extract our archive in D:\liferay. Our workspace will be in the same folder. Below is our file arborescence :

  • eclipse
  • workspace

    • Setting Liferay application server

    Get Liferay portal at following URL : http://www.liferay.com/downloads/liferay-portal/available-releases. You will get Liferay application bundled with a pre-configured application server. Most commonly used application server is Tomcat 7, so this is the one we will use here.

    Extract the archive to the same folder you put your IDE (D:\liferay in my case). Your aborescence should be now :

  • data
  • eclipse
  • tomcat-7.0.27
  • workspace

    • Now, in your IDE, click on the Servers view and open new server wizard link. Select Liferay, Inc., then Liferay v6.1 CE Server (Tomcat 7) and click on Next. On the next screen, browse to your application server directory (tomcat-7.0.27), and click on Finish.

    Importing tapestry libraries into your workspace

    Getting tapestry-liferay and tapestry-portlet could be useful before developping a Liferay portlet application using Tapestry. You can get them by using Import/Git/Projects from Git in your IDE, with the following URLs :

  • https://github.com/got5/tapestry5-portlet.git for tapestry-portlet
  • https://github.com/got5/tapestry5-liferay.git for tapestry-liferay

    • Do not forget to configure these two projects as Maven projects to handle their dependencies (right click on the project, Configure, and Convert to Maven Project.

    Tapestry5 Portlet allows you to run Tapestry5 web apps in a portal environment without worrying about the underlying portlet API or portlet concepts. Documentation is available at http://got5.github.com/tapestry5-portlet/.

    Tapestry5 Liferay is an Tapestry5 Portlet extension, which will help you creating new Tapestry portlets on Liferay.

    tapestry5-liferay contains a sub-project, tapestry-liferay-demo, which provides you lots of sample portlets, using tapestry5-jquery (https://github.com/got5/tapestry5-jquery) and KAWWA (https://github.com/got5/KAWWA).
